# Sokoban — AI 원샷 게임 제작 프로젝트

## 대회 규칙

| 규칙 | 내용 |
|------|------|
| **입력** | 프롬프트 텍스트만 가능. 사전 코드 작성 금지 |
| **실행** | AI가 코드 작성부터 실행, 에러 수정까지 자율 수행 |
| **인간 개입** | 불가. 심판이 시작하면 AI 혼자 끝까지 |
| **에러 수정** | 도중 실행 → 에러 수정 반복 가능 |
| **시각 검증** | AI가 직접 게임을 조작하고 스크린샷으로 검증 |
| **최종 목표** | `python main.py` 실행 시 에러 없이 게임이 동작하고, 레벨 1을 클리어할 수 있어야 함 |

### v1 → v2 수정 사항

1. 9개 불량 레벨을 BFS 솔버 검증 완료 레벨로 교체
2. "레벨을 새로 만들지 말 것" 절대 규칙 신설
3. par 값을 솔버 최적해로 정확히 설정
4. 자율 수정 절차에 "레벨 데이터 검증" 단계 추가
5. 인터페이스 시그니처 보강 (AnimState, SaveManager 상세화)
6. STATE_QUIT 처리 흐름 명시

## 기능 목록

- 20개 레벨 (Tutorial 5 / Easy 5 / Medium 5 / Hard 5)
- 4방향 이동 (방향키 / WASD)
- 박스 밀기, 클리어 판정
- Undo (Z) / Redo (Y) / Restart (R)
- 일시정지 메뉴 (ESC)
- 레벨 선택 화면 (잠금 해제 시스템)
- 별점 시스템 (par 기준 1~3성)
- 레벨 에디터 (마우스 타일 배치, 테스트 플레이)
- 세이브/로드 (JSON)
- 프로시저럴 사운드 (numpy)
- 파티클 이펙트 (스파클, 컨페티)
- 픽셀아트 스프라이트 (pygame.draw 코드 생성)
- 이동 보간 애니메이션 (lerp)
- 데드락 감지 (코너)
- --test 자동 검증 모드

## 실행 방법

```bash
pip install pygame numpy
python main.py          # 일반 실행
python main.py --test   # 자동 테스트 모드
```
